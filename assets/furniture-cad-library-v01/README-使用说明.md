# 住宅 CAD 家具图块库 V01

## 正式文件

- `furniture-library.json`：尺寸、净距、操作范围和来源等级。
- `blocks/`：每件家具的独立俯视 SVG 图块。
- `furniture-block-catalog-V01.svg/png`：图块目录预览。

## 使用规则

1. 图块尺寸单位统一为毫米，放入平面图时必须与户型采用同一比例。
2. 床的标注以床垫尺寸为主，实际图块外轮廓为床架占地。
3. 座椅、枕头、沙发靠背等用于表达使用方向，不另加方向箭头。
4. 操作范围与通道净距只在尺度校核图中显示，客户主平面默认隐藏。
5. 家电使用通用占位块时必须标记 `generic_placeholder`；确定型号后替换为产品专用块。
6. 图块不足时先扩充 JSON 和 SVG，经目录预览确认后才能用于方案。

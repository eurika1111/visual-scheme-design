# 客户需求收集模板 v1

使用位置：底图确认通过后、用户批准的方案方向生成前。

原则：每题都允许客户表达“不确定”“想看建议”“要看效果再决定”。不要逼客户一次性说死，模糊回答会进入 `unknowns` 或 `exploration_items`，用于后续方案对比和追问。

## 1. 常住人口与家庭结构

- 明确：1人 / 2人 / 三口之家 / 三代同住 / 有老人 / 有小孩 / 有宠物
- 模糊：暂时不确定未来居住变化 / 可能会有老人小住 / 未来可能有小孩
- 可补充：作息、在家办公、孩子年龄、宠物习惯

## 2. 必须保留的房间

- 明确：保留3卧 / 保留2卧+书房 / 需要儿童房 / 需要客房 / 需要多功能房
- 模糊：房间数量可以听建议 / 想看看空间利用率后再决定

## 3. 拆改接受度

- 明确：不拆墙 / 只接受轻微拆改 / 可局部拆改 / 可以较大胆调整
- 模糊：不排斥拆改，但要看效果和成本 / 先给我看看可能性

## 4. 厨房开放程度

- 明确：封闭厨房 / 半开放 / 开放厨房 / 想增强餐厨关系
- 模糊：开放不开放都可以，想看对比 / 担心油烟但喜欢通透感

## 5. 岛台态度

- 明确：想要岛台 / 可有可无 / 不要岛台
- 模糊：喜欢岛台感觉，但担心占地方 / 如果合适可以考虑

## 6. 收纳优先级

- 明确：普通收纳 / 强收纳 / 需要家政区 / 需要储物间 / 需要整墙柜
- 模糊：现在东西不算多但怕以后不够 / 想看不压抑的收纳方案

## 7. 生活重点

- 明确：亲子活动 / 居家办公 / 客餐厅社交 / 卧室舒适 / 家政效率 / 展示收藏
- 模糊：平时生活比较普通，希望整体更舒服 / 不知道怎么取舍，想看方案差异

## 8. 风格倾向

- 明确：现代 / 原木 / 奶油 / 极简 / 轻法 / 复古 / 侘寂
- 模糊：不要太冷 / 不要太复杂 / 喜欢干净温暖 / 先看布局，风格后定

## 9. 预算与施工风险

- 明确：低预算少施工 / 中等预算功能优先 / 效果优先预算可弹性
- 模糊：预算还没定 / 希望性价比高 / 可以先看高低两档

## 10. 不可动区域

- 明确：厨房不动 / 卫生间不动 / 承重墙不动 / 阳台不封 / 入户门不动
- 模糊：不清楚哪些能动，需要你标出风险
